This profile is reserved for:

- Unicode text shaping and bidi (FriBidi + HarfBuzz style),
- UTF-8 input to glyph runs (glyph IDs + per-glyph positions),
- stable metrics needed by layout engines.

The normative ABI will be added in a follow-up revision.
